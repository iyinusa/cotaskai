CoTaskAI Documentation

Welcome to the CoTaskAI documentation. This guide will help you understand how to install, configure, and make the most of the CoTaskAI browser extension.

Installation

CoTaskAI is available for major browsers through their respective extension stores:

Chrome

  1. Visit the Chrome Web Store
  2. Click "Add to Chrome"
  3. Confirm the installation when prompted

Firefox

  1. Visit the Firefox Add-ons Store
  2. Click "Add to Firefox"
  3. Confirm the installation when prompted

Edge

  1. Visit the Edge Add-ons Store
  2. Click "Get"
  3. Confirm the installation when prompted

Manual Installation (Developer Mode)

For advanced users who want to install the extension manually or for development purposes:


# Clone the repository
git clone https://github.com/iyinusa/cotaskai.git

# Navigate to the project directory
cd extension

# Install dependencies
npm install

# Build the extension
npm run build

Then follow these steps to install locally:

  1. Open your browser's extension management page:
    • Chrome: chrome://extensions
    • Edge: edge://extensions
    • Firefox: about:debugging#/runtime/this-firefox
  2. Enable Developer Mode
  3. Click "Load unpacked" (or "Load temporary add-on" in Firefox)
  4. Select the "dist" folder from the build output
Note: Manual installation requires the extension to be reloaded after browser restarts in development mode.

Quick Start Guide

Once installed, you can start using CoTaskAI right away:

  1. Access the extension: Click the CoTaskAI icon in your browser's toolbar
  2. Select text on any webpage: Highlight text to analyze it
  3. Use the context menu: Right-click on selected text to access quick actions
  4. Ask questions: Use the extension popup to ask questions about the current page

Text Analysis

CoTaskAI can analyze selected text in various ways:

How to use:

  1. Select any text on a webpage
  2. Right-click and choose "Analyze with CoTaskAI" or use the keyboard shortcut (Ctrl+Shift+A)
  3. Choose the analysis type from the dropdown menu

Available analysis types:

  • Summarize: Create a concise summary of the text
  • Explain: Get a simplified explanation of complex content
  • Find key points: Extract the main points from the text
  • Check facts: Verify factual claims in the text
  • Translate: Convert text to another language
Important: Text analysis features work best with moderate amounts of text (up to a few paragraphs). For longer content, consider using the summarization feature first.

Web Search Assistant

CoTaskAI can enhance your web searches by providing contextual information and additional insights:

Features:

  • Search suggestions: Get better search queries based on your initial search
  • Result enhancement: See additional context for search results
  • Quick answers: Get immediate answers for factual questions
